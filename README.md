# RMVision
致谢
首先感谢各高校开源代码为本套代码提供的参考，以及感谢2022赛季算法组师兄们的努力。秉承着开源精神，促进各战队间技术交流提升，我队决定本赛季开源本套代码，希望对其他战队提供一定的参考价值。

说明
本套代码是太原科技大学NewMaker战队2023赛季视觉代码开源，仅提供战队间技术交流使用，不得用于任何商业行为。

本套代码主要含有：装甲板识别和预测、数字图案识别、能量机关识别和预测，位姿解算击打等。
